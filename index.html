Wifi-QR-Poster

Generate a poster with password & QR Code for your Wifi network. Your guests then only need to scan it with their phone. No more "how's the password spelled?"-hassle!

When scanned with an Android phone, the Wifi network will be immediately recognized and you will be asked to connect with it. On iPhone/iOS, you will see the text and can copy/paste the password from it. Just try it out with the code aside.

Generated on https://georgjaehnig.github.io/wifi-qr-poster/
